## 一、AI 擅长题材（高匹配度）

### 🔥 都市系统流

**代表题材**：

- 神豪系统（花钱返利/消费返现）
- 任务系统（完成行为获得奖励）
- 技能树系统（升级解锁能力）
- 签到系统（每日打卡奖励）

**AI 优势原因**：

Text

1. 模板极度标准化：系统激活 → 新手任务 → 奖励结算 → 下一任务

2. 爽点可量化：数字增长天然具备"爽感"

3. 变种丰富：可快速混搭"系统+神医/鉴宝/外卖/直播"

**典型套路核心**：

- 系统绑定 → 完成简单任务 → 获得第一笔奖励
- 奖励用途明确 → 解决现实困境
- 对手嘲讽 → 打脸反转 → 收获更多奖励

**推荐指数**：⭐⭐⭐⭐⭐（上手最快、变种最多）

### 🔥 都市专业行业文

**代表题材**：

- 医生文（急诊/手术/中医）
- 律师文（法庭辩护/商业诉讼）
- 鉴宝/赌石（捡漏/打眼）
- 金融/投资（股市/期货/风投）
- 厨师/餐饮

**AI 优势原因**：

Text

1. 专业知识可检索整合

2. 行业规则有标准模板

3. 专业术语增加"真实感"

4. 升级路径清晰：初级从业者 → 资深专家 → 行业权威

**典型核心冲突**：

- 技术能力 vs 资源限制（没设备/没人脉）
- 规则正义 vs 潜规则（医疗黑幕/法律漏洞）
- 个人理想 vs 体制压迫

**推荐指数**：⭐⭐⭐⭐（专业性提升差异化）

### 🔥 玄幻/修仙升级流

**代表题材**：

- 传统玄幻（丹田/经脉/境界）
- 凡人流（资源匮乏型逆袭）
- 退婚流（废柴崛起）
- 系统签到修仙

**AI 优势原因**：

Text

1. 境界体系高度模板化（炼气→筑基→金丹...）

2. 升级目标明确（修为/法宝/功法）

3. 地图递进模式固定（宗门→秘境→大陆→仙界）

4. 对战公式化（属性克制/越级挑战）

**典型套路核心**：

- 资质测试 → 被判定为废柴
- 偶得机缘（戒指老爷爷/神秘功法/系统）
- 快速突破 → 打脸当初看不起自己的人
- 加入宗门 → 进入更高级地图

**推荐指数**：⭐⭐⭐⭐（框架成熟，篇幅可长可短）

### 🔥 穿越/重生都市

**代表题材**：

- 重生2000年代（利用信息差赚钱）
- 魂穿异世（都市背景）
- 历史穿越（清/唐/宋/明）
- 星际/未来穿越

**AI 优势原因**：

Text

1. "先知"优势天然制造爽点

2. 信息差利用模式固定

3. 原主关系网提供冲突素材

4. 重生前后对比强化戏剧性

**典型核心冲突**：

- 知道未来但无法直接改变（蝴蝶效应）
- 弥补前世的遗憾（情感线/复仇线）
- 身份错位（现代人思维 vs 古代/异世规则）

**推荐指数**：⭐⭐⭐⭐（情感冲突丰富）
