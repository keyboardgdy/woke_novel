# 可视化前端运行说明

## 开发模式

后端 API：

```bash
python -m app_server.main
```

前端开发服务器：

```bash
cd frontend
npm install
npm run dev
```

开发访问地址：

```text
http://127.0.0.1:5173
```

Vite 会把 `/api` 代理到 `http://127.0.0.1:8787`。

## 构建后运行

构建前端：

```bash
cd frontend
npm run build
```

启动后端：

```bash
python -m app_server.main
```

构建产物存在于 `frontend/dist` 时，FastAPI 会托管静态前端。

访问地址：

```text
http://127.0.0.1:8787
```

API 健康检查：

```text
http://127.0.0.1:8787/api/v1/health
```

## 命令行一键打开

项目根目录提供了 `woke.bat`：

```bat
woke
```

它会静默启动本地 FastAPI 后端，并打开浏览器访问：

```text
http://127.0.0.1:8787
```

要在任意命令行直接输入 `woke`，把项目根目录加入当前用户 PATH：

```powershell
$path = [Environment]::GetEnvironmentVariable("Path", "User")
if (($path -split ";") -notcontains "D:\woke_novel") {
  [Environment]::SetEnvironmentVariable("Path", "$path;D:\woke_novel", "User")
}
```

重新打开一个终端后生效。

## 当前 MVP 范围

- 项目列表与搜索。
- 新建项目向导。
- 完整执行、断点续跑、实时 SSE 日志。
- 项目文件树与 Markdown/文本/JSON 读取。
- 章节列表、字数统计、导出前检查。
- MD/TXT/EPUB 导出，其中 EPUB 需要 `ebooklib`。
- 历史日志列表与 tail 查看。
- 设置与环境检测。

## 注意事项

- 服务按本地单用户模型设计，默认绑定 `127.0.0.1:8787`。
- 前端不直接读取本地文件，所有文件访问由 API 做项目路径约束。
- 真实运行会调用现有 `run_workflow.py`，并继续使用 Claude CLI / Codex CLI 的现有自动化参数。
- `run_workflow.py single` 当前仍不支持 `act_num`、`round_num`、`option_index`，因此循环步骤的精确上下文单步重跑还属于后续增强。
