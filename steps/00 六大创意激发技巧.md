| 技巧                    | 操作步骤                                | 示例  |
| --------------------- | ----------------------------------- | --- |
| **1️⃣ 反转假设**          | 将「创意种子」的**核心假设**倒置，问："如果相反会怎样？"     |     |
| **2️⃣ 交叉规则**          | 把**两条世界观规则**强行叠加，观察冲突。              |     |
| **3️⃣ 角色冲突升级**        | 将 **角色欲望** 与 **世界观限制** 置于 **极端情境**。 |     |
| **4️⃣ 随机词根 + 主题标签**   | 用 **随机词**或 **主题标签**注入每段转折。          |     |
| **5️⃣ "Yes, And" 扩张** | 对每个已有的创意点 **接受并扩展**，不加否定。           |     |
| **6️⃣ 约束驱动**          | 设定 **硬性限制**，迫使**创意解决方案**出现。         |     |
