# 故事演化核心原则（所有剧情规划阶段必须遵守）

剧情推进必须遵循“因果驱动、人物驱动、渐进升级”原则。

故事的发展应来自：
- 角色欲望与性格缺陷
- 利益冲突与信息差
- 已有矛盾持续发酵
- 事件后果不断累积
- 局势压力逐步升级

所有冲突、关系变化、情绪爆发与重大转折，
都必须具备明确铺垫与演化过程。

优先制造：
- 潜在危机
- 压力积累
- 隐性裂痕
- 局势恶化
- 不可逆后果

而不是过早进入高烈度爆点。

禁止：
- 为了刺激强行升级剧情
- 角色降智推动剧情
- 无代价收益
- 无铺垫反转
- 情绪与关系瞬间极端化
- 世界规则失效
- 连续高强度爆点轰炸

故事应始终保持：
“角色做出选择 → 产生代价 → 改变局势 → 逼迫下一轮选择”
的连续因果链。
