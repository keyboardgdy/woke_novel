设计生成一份写作指南，用于指导AI生成小说正文。

【设计原则】
1. 指南不重复剧情，剧情由剧情文件提供；指南只做执行层指令
2. 每条指令必须可执行——AI读完即知道该写什么、不该写什么，而非给方向但留解释空间
3. 风格用指纹不用标签（不说"古典雅致"，要给句长/词汇/修辞密度的具体刻度）
4. 情绪用纹理不用归类（不说"沉重压抑"，要写"克制的热/慢性的冷/闷在骨头里的焦灼"）
5. 禁忌要靶向不要泛泛（针对本故事的AI典型退化模式，不是通用写作常识）
6. 
【输出要求】
- 优先质量，不追求篇幅
- 指令要具体到可以照着写，不要留模糊空间
- 禁止项必须定制化，基于本故事的类型/人物/风格特征
- 严格使用下方章节结构
写入：{guides}/写作指南v{round}.md
## 一、叙事坐标

1.1 定位锚点

- 当前位于第几幕第几场，引用剧情文件中的场景编号
- 前一场已落定的事件（一句话），下一场将接的入口（一句话）
- 本段在整体弧线中的功能位置（蓄力 / 转折 / 释放 / 回落）

1.2 场景锚点

- 具体地点（精确到房间 / 街道 / 时辰）
- 3-5个可作为描写支点的物件（标注功能：信息载体 / 情绪载体 / 氛围载体）
- 感官锚点（至少覆盖两个通道：视觉 / 听觉 / 嗅觉 / 触觉 / 味觉）

1.3 时间锚点

- 故事内时间（具体到时辰或事件发生后的第几天）
- 章节内时间跨度（这一段覆盖几分钟 / 几小时 / 几天）

---

## 二、风格指纹

2.1 句式特征

- 主用句长区间（短句为主 / 长短交替 / 长句为主）
- 句式偏好（是否使用文言句式 / 欧化句式 / 口语断句 / 倒装）
- 段落密度（一段一句 / 一段多层 / 场景内整段不分行）

2.2 词汇域

- 时代语域（古白话 / 近代白话 / 现代口语 / 混合）
- 职业 / 地域方言特征（是否带特定行话、方言词、术语）
- 禁用词方向（描述倾向，如"避免现代网络用语""避免古风过度堆砌"）

2.3 修辞密度

- 每千字的修辞事件数（比喻 / 通感 / 拟人等）
- 修辞类型偏好（意象型 / 反差型 / 感官型 / 哲思型）
- 是否有特殊修辞（反复 / 留白 / 断句 / 意识流片段）

2.4 情绪纹理

- 本段主导情绪的质感（不是"什么调"，是"什么质感"，如：克制的热、慢性的冷、闷在骨头里的焦灼）
- 主导情绪 + 1-2个变奏点（在哪个事件上情绪会发生转向）

---

## 三、人物声音

3.1 主要人物

- 说话方式（句尾习惯 / 用词偏好 / 语速节奏 / 是否经常欲言又止）
- 行为特征（标志性小动作 / 对特定物件的态度 / 身体语言模式）
- 本段内的心理状态（写"他在两个选择之间反复掂量，但开口时却选了第三个"，不写"他很纠结"）

3.2 关系动态

- 当前两人关系的实质（表层 / 里层是否一致）
- 本段关系是推进 / 凝固 / 倒退
- 关系变化的触发事件与触发后的状态

3.3 配角刻度

- 配角功能定位（信息传递 / 氛围烘托 / 对照 / 伏笔 / 工具人）
- 描写深度（有名有姓有几句台词 / 纯背景 / 一笔带过）
- 是否需要在本段建立配角的辨识度

---

## 四、情节执行

4.1 必达事件

- 引用剧情文件中的具体节点编号或场景名（不复述剧情内容）
- 每个节点的执行优先级：
    - 必须落地（核心事件，不可省略）
    - 可灵活处理（可调整顺序或合并）
    - 可裁剪（视篇幅可舍去）

4.2 信息密度控制

- 哪些信息要在本段首次透露（必须让读者明确接收到）
- 哪些信息要延后（读者可能猜到但不能写明）
- 哪些信息要模糊处理（留疑不解，给读者想象空间）

4.3 情绪弧线

- 起点状态（角色入场时的情绪）
- 转折触发点（哪个事件 / 哪句话 / 哪个细节会引发情绪转向）
- 终点状态（角色出场的情绪）
- 延迟释放点（情绪压在哪个位置故意不落，让它延后到下段）

4.4 伏笔与回收

- 引用剧情文件中已标注的伏笔编号（哪些要在本段回收）
- 本段新埋的伏笔（具体载体：物件 / 对话 / 细节动作，不写大纲只写载体）
- 伏笔的伪装程度（一眼可见 / 细想可察 / 完全隐蔽）

4.5 落幕方式

- 本段结尾的悬念形态（未说完的话 / 未揭开的真相 / 未到达的人 / 情绪反刍 / 场景骤停）
- 末句的情绪走向（上扬 / 下沉 / 平收 / 反转）
- 末句的句式特征（短促 / 反问 / 未完成 / 意象收束）

---

## 五、节奏控制

5.1 篇幅与结构

- 字数区间（上下限）
- 内部节拍数与每个节拍的大致字数占比

5.2 段落类型与节奏分配

- 动作段 / 心理段 / 对话段 / 留白段 在本段中的位置与配比
- 慢镜头 / 快进 / 留白 的具体落点
- 张弛比例（如：3拍紧 + 1拍松 / 整体张 / 整体弛）

---

## 六、靶向禁忌

6.1 禁忌清单

- 针对本故事定制，每条一句话，具体到可识别
- 不留通用例子，不重复AI默认的写作常识
- 形式：禁止 + 原因（为什么这条对本故事特别重要）

6.2 高风险位置

- 本段哪里最容易写崩（节奏 / 情感 / 信息 / 风格）
- 每个风险点给一句预警（写到这里要注意什么）
