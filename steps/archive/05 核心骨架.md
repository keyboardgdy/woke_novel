## 生成 **核心骨架**（Spine Beats）

结合 **六大创意激发技巧** 生成每个节点：
- **反转假设**：将当前节点的默认假设倒置
- **交叉规则**：叠加两条世界观规则制造冲突
- **角色冲突升级**：将角色欲望与世界规则限制置于极端情境
- **随机词根 + 主题标签**：注入随机词或主题标签带来惊喜转折
- **"Yes, And" 扩张**：接受并扩展已有创意点
- **约束驱动**：设定硬性限制迫使创意解决方案出现

### **主线节点（Major Beats）**
**思考方式**：每个节点都是**对核心问题的"回答/升级"**，同时必须牵动**角色内在冲突**和**世界规则的冲突**
⚠️ 核心原则：节点叙事粒度统一**一个主线节点 = 一个完整的「戏剧动作」**

| 节点                          | 目的                            |
| --------------------------- | ----------------------------- |
| **Hook（开场钩）**               | 把「创意种子」瞬间显形，引发观众好奇            |
| **Setup（铺垫）**               | 展示 **角色档案** 与 **世界规则** 的日常状态  |
| **Inciting Incident（催化点）**  | 把「创意种子」强行植入，打破平衡              |
| **First Plot Point（第一转折）**  | 把主角推离原有轨迹，进入 **新情境**          |
| **Midpoint（中间点）**           | 把冲突 **从外到内**（或反之）转移，出现全新信息或力量 |
| **Second Plot Point（第二转折）** | 危机加剧，所有线索交叉，冲突逼近极限            |
| **Crisis（危机点）**             | 必须做出 **两难抉择**，往往对应角色 **核心价值** |
| **Climax（高潮）**              | 冲突的 **终极对决**（可以是内在、外在、概念层面）   |
| **Resolution（收束）**          | 给出 **答案**（对核心问题的最终回应），并留下余韵   |
**产出**：一张 **骨架表格**（节点 + 核心冲突 + 角色变化 + 世界观冲击），每行都是后续细节的"锚点"。

### 细化 **次要节点**（Sub‑Beats）

- **从每个主节点** 向外展开 2‑4 条 **次要情节线**（支线、角色弧、背景设定）。
- 每条次要线要 **映射** 主线冲突（如：配角的目标被世界规则限制，主角的目标被配角的选择影响）。
- 用 **"如果…会…？"** 的逆向思考为每条支线设计 **惊喜转折**。
