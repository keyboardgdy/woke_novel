| Technique | Operating Steps | Example |
| --- | --- | --- |
| 1. Reverse the Assumption | Invert the core assumption of the creative seed and ask: "What if the opposite were true?" | |
| 2. Cross the Rules | Force two worldbuilding rules to overlap and observe the conflict. | |
| 3. Escalate Character Conflict | Put character desire and world limitations into an extreme situation. | |
| 4. Random Root + Theme Tag | Inject a random word root or theme tag into each turning point to create surprise. | |
| 5. "Yes, And" Expansion | Accept each existing idea and extend it instead of negating it. | |
| 6. Constraint-Driven Design | Set a hard limitation that forces a creative solution to appear. | |
